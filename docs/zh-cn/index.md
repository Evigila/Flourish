# Flourish

Flourish 是面向 WPF 应用的桌面应用组合库。它提供基于 .NET Host 的启动流程、Shell 配置、导航注册、动态工具栏、状态栏集成和动效选项。

这套文档由 DocFX 从源码项目和 XML 文档注释生成。中文站点会复用同一份 API 签名数据，并通过 overwrite 文件提供中文 API 说明。API 参考刻意只公开 `AcksheedSys.Flourish.Abstract` 契约。
